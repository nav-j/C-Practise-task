## Task: Bitwise Operators Practice

Analyze the following C programs and **write the final output** for each.
Show **binary steps** for full marks.

---

### **Task 1: XOR Assignment**

```c
int main() {
    int x = 12;
    x ^= 5;
    printf("%d", x);
    return 0;
}
```

---

### **Task 2: Right Shift Assignment**

```c
int main() {
    int x = 20;
    x >>= 2;
    printf("%d", x);
    return 0;
}
```

---

### **Task 3: Left Shift Assignment**

```c
int main() {
    int x = 7;
    x <<= 4;
    printf("%d", x);
    return 0;
}
```

---

### **Task 4: Combined Operations**

```c
int main() {
    int x = 10;
    x <<= 1;
    x ^= 3;
    printf("%d", x);
    return 0;
}
```

---

### **Task 5: Predict the Output**

```c
int main() {
    int x = 18;
    x >>= 3;
    x <<= 2;
    printf("%d", x);
    return 0;
}
```

---

##  Instructions

* Convert numbers to **binary**
* Apply the operator step by step
* Convert the result back to **decimal**
