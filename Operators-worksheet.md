## Section A: Arithmetic Operators

**Q1.** What will be the output?

```c
int a = 8, b = 3;
printf("%d", a + b * 2);
```

**Q2.** Evaluate the result:

```c
int x = 10;
printf("%d", x / 4);
```

**Q3.** Find the output:

```c
int a = 15;
printf("%d", a % 4);
```

---

## Section B: Assignment Operators

**Q4.** What is the final value of `x`?

```c
int x = 5;
x += 6;
printf("%d", x);
```

**Q5.** Predict the output:

```c
int a = 4;
a *= 3;
printf("%d", a);
```

---

## Section C: Comparison Operators

**Q6.** What will be printed?

```c
int a = 7, b = 5;
printf("%d", a > b);
```

**Q7.** Find the output:

```c
int x = 10;
printf("%d", x == 8);
```

---

## Section D: Logical Operators

**Q8.** What is the output?

```c
printf("%d", 5 > 3 && 2 < 1);
```

**Q9.** Predict the output:

```c
int a = 0;
printf("%d", !a);
```

---

## Section E: Mixed Operators

**Q10.** Evaluate the expression:

```c
int x = 6;
printf("%d", x > 5 && x < 10);
```

**Q11.** What will be printed?

```c
int a = 5;
a += 2 * 3;
printf("%d", a);
```

**Q12.** Find the output:

```c
int x = 4, y = 8;
printf("%d", x < y || y == 10);
```
